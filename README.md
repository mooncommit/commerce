# 🛒 실시간 커머스 플랫폼


---

## 😎 프로젝트 소개

필수과제까지 끝낸 콘솔 기반 커머스 플랫폼입니다.

카테고리별로 상품을 둘러보고 원하는 상품을 조회할 수 있습니다.
객체지향 설계를 적용해서 각 클래스가 자기 역할만 담당하도록 만들었습니다!

---

## 📁 프로젝트 구조
```
src
├── Main.java            # 프로그램 시작
├── Product.java         # 개별 상품 관리
├── Category.java        # 카테고리별 상품 묶음 관리
├── CommerceSystem.java  # 전체 플랫폼 로직 및 사용자 입력 처리
└── Customer.java        # 고객 정보 관리
```

---

## ✅ 구현 기능

- 카테고리별 상품 목록 출력
- 카테고리 선택 및 상품 조회
- 상품 상세 정보 출력 (상품명, 가격, 설명, 재고)
- 잘못된 입력값 예외 처리
- 프로그램 종료 및 뒤로가기

---

## 💻 실행 화면
```
[ 실시간 커머스 플랫폼 메인 ]
1. 전자제품
2. 의류
0. 종료      | 프로그램 종료
입력 : 1

[ 전자제품 카테고리 ]
1. Galaxy S25 | 1,200,000원 | 최신 안드로이드 스마트폰
2. iPhone 16  | 1,350,000원 | Apple의 최신 스마트폰
3. MacBook Pro | 2,400,000원 | M3 칩셋이 탑재된 노트북
4. AirPods Pro |   350,000원 | 노이즈 캔슬링 무선 이어폰
0. 뒤로가기
입력 : 2

선택한 상품 : iPhone 16 | 1,350,000원 | Apple의 최신 스마트폰 | 재고 : 10개
```

---

## 📦 클래스 설계

| 클래스 | 역할 |
|---|---|
| `Main` | 객체 생성 및 프로그램 시작 |
| `Product` | 상품명, 가격, 설명, 재고수량 관리 |
| `Category` | 카테고리 이름, 상품 리스트 관리 |
| `CommerceSystem` | 사용자 입력 처리 및 전체 흐름 관리 |
| `Customer` | 고객명, 이메일, 등급 관리 |
